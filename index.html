VR AR MR

Immersive Media is attached to many acronymes.

Due to those technologies being new, the following definitions are still a matter of debate.

Virtual reality (VR)

  • Fully CG
  • Requires a special headset for now
  • The user is inside the virtual world
  • VR apps & games

Augmented Reality (AR)

  • Most of time it is CG elements projected in the real world
  • Requires sensors to constantly capture the environment
  • The user still can distinguish real world from virtual one
  • Pokemon GO

Mixed reality (MR)

  • Real world and virtual world are tightly connected
  • Sensors are used on both sides
  • Real time interaction between both worlds
  • Hololens

How it works

The content is rendered twice, once per eye.

Using a perspective angle for each eye.
